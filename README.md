# 錦鯉シミュレーション (Nishikigoi Simulation)

ブラウザで動く錦鯉育成・交配シミュレーションゲームです。美しい錦鯉を育て、交配させて新しい品種を生み出し、図鑑を完成させましょう。

## セットアップと実行

1. リポジトリをクローンします。
2. `index.html` をブラウザで開きます。

## ゲームの特徴

### 🐟 錦鯉の育成と交配
- 池で錦鯉を飼育し、2匹を選択して交配させます
- DNAシステム（基本色、模様色、模様密度）による遺伝
- 突然変異で新しい色や模様が出現
- **成長システム**: 稚魚→若魚→成魚の3段階成長
- **成魚のみ交配可能**: 稚魚・若魚は交配不可

### 🍚 餌やりシステム
- **餌やりモード**: 右下の「🍚 餌やり」ボタンで餌やりモードを切り替え
- **支払い方式**: 餌を1回投げるごとに1万円が自動的に支払われる
- **成長促進**: 餌を食べると成長進捗が20%増加（5個で成魚）
- **リアルタイム反応**: 鯉が餌に近づく速度・方向転換を強化（通常の2.5倍まで速度上昇）

### 🏆 評価システム
- 錦鯉のサイズ、色、模様、世代に基づいて評価
- **成長ステージ考慮**: 稚魚は評価額30%、若魚は60%、成魚は100%
- ランク付け（S, A, B, C）
- 評価額に応じた売却価格

### 📖 図鑑システム
- **品種コレクション**: 21種類の錦鯉品種を発見（ピンク・銀色追加）
- **歴代記録**: 最高評価額と最大サイズの記録
- **名品アルバム**: Sランク・Aランクの錦鯉を記録

### 🏞️ 池の管理
- ポイントを使って池を拡張
- 最大10匹まで飼育可能
- 自動収入システム（観客が訪れて収入を得る）

### 💾 データ保存
- localStorageを使用した自動セーブ/ロード
- ゲーム進行の継続が可能

### 📱 レスポンシブデザイン
- PC、タブレット、スマートフォン対応
- タッチ操作対応

## 開発者向け情報
ゲームはオブジェクト指向設計で実装されており、主要なクラスは以下の通りです：
- `Koi`: 錦鯉の描画、更新、遺伝情報管理
- `FloatingText`: 浮遊テキストエフェクト
- `Visitor`: 観客の描画と動作

遺伝システムは複数の突然変異ルールを含み、世代を重ねるごとに希少な色（金、紫など）が出現する確率が上がります。
